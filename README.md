# Challenge

Challenge app React Native

## Requirements
To build this project in your machine you will need.
* Node 16 or newer
* yarn

## Instalation 
Clone the repo to a working directory:
```bash
git clone https://github.com/aalvarez10/credit-bank-tata.git
``` 

Go to root folder:
```bash
yarn install
```

## Run App

```bash
yarn android
```

## Run Test
```bash
yarn Test
```
